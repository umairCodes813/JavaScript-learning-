var

“We should not use the var type because it declares variables globally. If we declare a var inside a block and print it, and then declare another var outside the block and print it, both times the value from the block will be printed.”

let

“We should use let most of the time because it declares variables locally. The values in both places are different, so they will be printed separately. Also, in let , we cannot re-initialize the same variable again to change its value.”

const

“When we use const , it means that this value cannot be changed throughout the entire program. We mostly use const for variables whose values we do not want to change in the whole program. We should use const more often because its advantage is that we can later change it to let if needed.”
